CASE — 001 Overall Risk: Critical

Access Control Failure Leading to System Compromise

An IDOR vulnerability in a network capture web application exposed packet files belonging to other users. Analysis revealed cleartext FTP credentials, which were reused to gain SSH access. A misconfigured Linux capability (cap_setuid on Python 3) then enabled full privilege escalation to root — demonstrating how three independent control failures compound into a complete compromise chain.
